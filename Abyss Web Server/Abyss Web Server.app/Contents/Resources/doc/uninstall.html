Appendix E. Uninstalling The Software

To remove the software from your computer, delete the following files:

  • The executable file Abyss Web Server.

  • The configuration file abyss.conf.

  • The persistency file persist.data.

  • The MacOS X application preferences file com.aprelium.AbyssWebServer.plist located in your Home folder inside the sub-folder Library/Preferences.
