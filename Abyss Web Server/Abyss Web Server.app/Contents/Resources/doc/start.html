Getting the software

If you already have the Abyss Web Server installation package, skip this section.

To download Abyss Web Server, browse http://www.aprelium.com/downloads. Follow the provided instructions and choose the correct version of the software suitable to your computer and to the operating system you are using. Once done, the browser displays a dialog box asking where to save the downloaded file. Choose a directory and validate.
