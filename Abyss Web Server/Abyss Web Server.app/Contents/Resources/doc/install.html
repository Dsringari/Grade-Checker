Installing/Upgrading the software

  • Double-click on the downloaded package icon. A new volume containing the installation package of Abyss Web Server will be mounted and available on the desktop.

  • Double-click on the mounted installation volume icon to open it.

  • For a first time installation, copy the folder Abyss Web Server somewhere on your hard disk. A suggested location is the Application folder.

  • If you are upgrading, stop Abyss Web Server, open the folder where the old version of Abyss Web Server is installed, delete the application file of Abyss Web Server and replace it with the new version of this file you will find in the installation package inside the folder Abyss Web Server. Copy also the new lang subfolder over the already installed one to update language files.

From now on, <Abyss Web Server directory> will refer to the folder where the software was copied (installed).
